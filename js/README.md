# Gilded Rose

This is the Gilded Rose kata in JavaScript

## Getting started

Install dependencies

```sh
npm install
```

## Running program

Run default:

```sh
npm start
```

To run with a certain number of days, simply add a number to the end of the command.

For example, to run 5 days:

```sh
npm start 5
```

And to run 25 days:

```sh
npm start 25
```
